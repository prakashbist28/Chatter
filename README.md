                       CHATTER
 
Chatter is a full stack chat application, It lets users to register, login and then chat with each other. It is implemented with the help of MERN stack, socket.io


Technologies used:
•	React js
•	Node js
•	Express
•	Mongo db
•	Bcrypt
•	Socket.io
